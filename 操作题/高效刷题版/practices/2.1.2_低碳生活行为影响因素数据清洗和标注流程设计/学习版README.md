## 一、代码题目要求（来自刷题模板）

1. 读取 `大学生低碳生活行为的影响因素数据集.xlsx`，查看前 5 行。
2. 删除缺失值所在行，并记录处理前后行数与删除行数。
3. 删除重复值并统计删除数量。
4. 对数值字段（题面给出的“月生活费”量表字段）做标准化。
5. 选择题目指定 8 项行为/背景特征，将 `低碳行为积极性` 设为目标变量。
6. 按 8:2 划分训练集与测试集。
7. 保存 `2.1.2_cleaned_data.csv`，并准备 `2.1.2.docx` 与 `2.1.2.html`。

## 三、文本题（2.1.2.docx）作答要点

### 数据清洗规范

- 先做字段核验，确保问卷列名与题面特征项一一对应。
- 缺失值处理采用删行法，并记录“处理前行数、处理后行数、删除行数”。
- 重复样本统一去除，避免重复问卷影响行为分布。
- 数值量表字段标准化，保证不同量纲可比较。
- 清洗后进行完整性复核（缺失值、重复值、字段类型）。

### 数据标注规范

- 明确 `低碳行为积极性` 为唯一目标变量。
- 分类特征建议进行编码（如独热编码）后再入模。
- 划分比例固定 8:2，随机种子固定，保证可复现。
- 训练和测试数据应来自同一清洗后的数据版本。
- 输出文件命名严格遵循题面要求。

## 四、易错点速记

- 把问卷题号列名抄错，导致特征选择报错。
- 仅删除缺失值，不统计删除前后样本数。
- 忽略重复样本处理。
- 保存文件名写成临时名，不符合提交要求。
