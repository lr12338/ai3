## 三、文本题（2.1.4.docx）作答要点

### 数据清洗规范

- 日期字段统一标准格式（`yyyy-mm-dd`），保证计算口径一致。
- 关键标识列命名规范化（`病人ID -> 患者ID`），便于后续治理。
- 派生字段应有明确定义：诊断延迟=诊断日期-就诊日期，病程=当前日期-诊断日期。
- 异常数据剔除规则需可解释（负病程、负延迟、年龄上限）。
- 去重、归一化后需再次核验数据范围与统计描述。

### 数据标注规范

- 明确可用于建模或分析的核心字段及其业务含义。
- 分类字段（疾病类型、治疗结果、严重程度）需统一枚举口径。
- 可视化图表标题、坐标轴含义与数据版本保持一致。
- 图表用于发现模式，不应替代字段级标注说明。
- 输出文件命名、路径、格式与题面一致。

## 四、易错点速记

- `病程` 计算基准日期混乱，导致结果为负值。
- 归一化前未去除明显异常值，压缩了正常区间。
- 中文字体路径写死 Windows 路径，在其他系统无法显示。
- 图表画出后未结合文本题说明其业务意义。
