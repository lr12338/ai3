# 3.2.4 花朵智能识别系统交互流程设计

## 学习建议

- 先打开 `exam_preview.html` 看题面，再做 `example.ipynb`。
- 做完后对照原始答题素材检查保存格式、命名要求和答题结构。

## 文件说明

- `answer_template.docx`：原始答题卷/文字题模板
- `example.ipynb`：主练习 Notebook
- `data/flower-detection.onnx`：题目配套数据或模型资源
- `data/flower_test.png`：题目配套数据或模型资源
- `data/labels.txt`：题目配套数据或模型资源
- `exam_preview.html`：考试平台模拟界面
