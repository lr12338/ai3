# 模拟试卷索引（5套）

## 固定组卷模板（每套100分）

1. 业务分析（1.1 或 1.2）25分（抽一）
2. 智能训练 - 数据处理规范制定（2.1）15分（必考）
3. 智能训练 - 算法测试（2.2）20分（必考）
4. 智能系统设计 - 智能系统监控与优化（3.1）15分（必考）
5. 智能系统设计 - 人机交互流程设计（3.2）20分（必考）
6. 培训与指导（4.1 或 4.2）5分（抽一）

## 题号映射规则

- `1.1.x`：业务流程设计（代码题）
- `1.2.x`：业务模块效果优化（文本题）
- `2.1.x`：数据处理规范制定（操作题）
- `2.2.x`：算法测试（操作题）
- `3.1.x`：系统监控与优化（文本题）
- `3.2.x`：人机交互流程设计（文本/流程设计题）
- `4.1.x`：培训大纲编写（抽一）
- `4.2.x`：数据采集与处理指导（抽一）
