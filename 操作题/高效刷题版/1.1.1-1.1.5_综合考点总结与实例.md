## 一、五题本质是同一题型

这 5 题本质都属于：**业务数据采集 -> 数据清洗/审核 -> 分组统计 -> 导出结果**。

无论场景怎么变，核心都在考：

1. `pd.read_csv()` 读取数据  
2. `dropna()/fillna()` 缺失值处理  
3. `astype()` 类型转换  
4. `between()/比较运算/布尔索引` 合理性筛选  
5. `np.where()` 打标签（高低风险、是否异常）  
6. `groupby()/value_counts()` 统计  
7. `pd.cut()` 分箱统计（年龄段、BMI段等）  
8. `to_csv(index=False)` 导出结果

---

## 二、通用答题模板（必背）

```python
import pandas as pd
import numpy as np

# 1) 读取
data = pd.read_csv("data/xxx.csv")
print(data.head())

# 2) 清洗
data = data.dropna()  # 或按题意 fillna  .ffill() .bfill()
data["A"] = data["A"].astype(int)
data["B"] = data["B"].astype(float)

# 3) 合理性筛选（多条件）
data = data[
    (data["A"].between(low, high)) &
    (data["B"] > 0) &
    (data["C"].between(c1, c2))
]

# 4) 打标签（可选）
data["Flag"] = np.where(condition, "高风险", "低风险")

# 5) 统计
cnt = data["某分类列"].value_counts()
grp = data.groupby("分组列")["指标列"].mean()

# 6) 分箱统计（可选）
bins = [18, 26, 36, 46, 56, 66, np.inf]
labels = ["18-25", "26-35", "36-45", "46-55", "56-65", "65+"]
data["AgeGroup"] = pd.cut(data["Age"], bins=bins, labels=labels, right=False)
age_cnt = data["AgeGroup"].value_counts().sort_index()

# 7) 导出
data.to_csv("cleaned_xxx.csv", index=False)
print("数据处理完成")
```

## 三、高频 API 速记（考试最常用）

- `data.isnull().sum()`：每列缺失值数量  
- `data.duplicated().sum()`：重复行数量  
- `series.between(a, b)`：区间判断，等价于 `>=a and <=b`  
- `data[条件]`：布尔筛选  
- `np.where(条件, 真值, 假值)`：标签/异常标记  
- `groupby(...).agg(["count", "mean"])`：多统计指标  
- `value_counts()`：分类计数  
- `pd.cut(..., right=False)`：左闭右开分箱  
- `fillna(method="ffill")` + `fillna(method="bfill")`：前后值填补  
- `drop(columns=[...])`：删除辅助列  
- `to_csv(..., index=False)`：导出且不保留索引

## 四、五题对应实例（按场景记忆）

## 1) 1.1.1 智能医疗（风险标签 + BMI/年龄分段）

核心动作：

- `住院天数 > 7` 标记高风险患者  
- 统计高/低风险人数及占比  
- 用 `pd.cut` 做 `BMI` 分段和年龄分段  
- 计算各区间高风险比例与患者数量

关键写法：

```python
data["RiskLevel"] = np.where(data["HospitalDays"] > 7, "高风险患者", "低风险患者")
risk_counts = data["RiskLevel"].value_counts()
high_risk_ratio = risk_counts["高风险患者"] / len(data)

data["BMIRange"] = pd.cut(data["BMI"], bins=[0, 18.5, 24, 28, np.inf],
                          labels=["偏瘦", "正常", "超重", "肥胖"], right=False)
bmi_risk_rate = data.groupby("BMIRange")["RiskLevel"].apply(lambda x: (x == "高风险患者").mean())
```

## 2) 1.1.2 智能农业（分组统计 + 异常标记 + 缺失填补）

核心动作：

- 统计每种传感器数量和均值  
- 按 `Location + SensorType` 统计温湿度均值并 `unstack()` 展开  
- 用业务阈值标记异常（温度、湿度）  
- 用 `ffill + bfill` 处理缺失并导出

关键写法：

```python
sensor_stats = data.groupby("SensorType")["Value"].agg(["count", "mean"])

location_stats = (
    data[data["SensorType"].isin(["Temperature", "Humidity"])]
    .groupby(["Location", "SensorType"])["Value"]
    .mean()
    .unstack()
)

data["is_abnormal"] = np.where(
    ((data["SensorType"] == "Temperature") & ((data["Value"] < -10) | (data["Value"] > 50))) |
    ((data["SensorType"] == "Humidity") & ((data["Value"] < 0) | (data["Value"] > 100))),
    True, False
)
```

## 3) 1.1.3 金融信用评估（完整性审核 + 合理性审核 + 清洗）

核心动作：

- 缺失/重复值统计（完整性）  
- 年龄、收入、贷款额度、信用分合理性审核（多布尔列）  
- `all(axis=1)` 合并审核结果  
- 仅保留 `is_valid=True` 的记录并删除审核辅助列

关键写法：

```python
missing_values = data.isnull().sum()
duplicate_values = data.duplicated().sum()

data["is_age_valid"] = data["Age"].between(18, 70)
data["is_income_valid"] = data["Income"] > 2000
data["is_loan_amount_valid"] = data["LoanAmount"] < (data["Income"] * 5)
data["is_credit_score_valid"] = data["CreditScore"].between(300, 850)

data["is_valid"] = data[
    ["is_age_valid", "is_income_valid", "is_loan_amount_valid", "is_credit_score_valid"]
].all(axis=1)

cleaned_data = data[data["is_valid"]].drop(
    columns=["is_age_valid", "is_income_valid", "is_loan_amount_valid", "is_credit_score_valid", "is_valid"]
)
cleaned_data.to_csv("cleaned_credit_data.csv", index=False)
```

## 4) 1.1.4 电商用户行为（类型转换 + 异常筛选 + 标准化 + 分组统计）

核心动作：

- 删除缺失值并转类型（年龄/金额/评分）  
- 按年龄、消费金额、评分做有效数据筛选  
- 对 `PurchaseAmount` 和 `ReviewScore` 做标准化  
- 统计购买类别人数、性别平均消费、年龄段人数

关键写法：

```python
data = data.dropna()
data["Age"] = data["Age"].astype(int)
data["PurchaseAmount"] = data["PurchaseAmount"].astype(float)
data["ReviewScore"] = data["ReviewScore"].astype(int)

data = data[
    (data["Age"].between(18, 70)) &
    (data["PurchaseAmount"] > 0) &
    (data["ReviewScore"].between(1, 5))
]

pa_m, pa_s = data["PurchaseAmount"].mean(), data["PurchaseAmount"].std()
data["PurchaseAmount"] = (data["PurchaseAmount"] - pa_m) / pa_s
```

## 5) 1.1.5 智能交通（清洗 + 合理性审核 + 多指标统计）

核心动作：

- 年龄、车速、距离、时长做类型转换和区间筛选  
- 反向筛选不合理数据 `~(条件)` 做审核输出  
- 统计交通事件次数  
- 按性别统计平均车速/距离/时长  
- 按年龄段统计驾驶员人数

关键写法：

```python
data = data.dropna()
data["Age"] = data["Age"].astype(int)
data["Speed"] = data["Speed"].astype(float)
data["TravelDistance"] = data["TravelDistance"].astype(float)
data["TravelTime"] = data["TravelTime"].astype(float)

valid_mask = (
    data["Age"].between(18, 70) &
    data["Speed"].between(0, 200) &
    data["TravelDistance"].between(1, 1000) &
    data["TravelTime"].between(1, 1440)
)

unreasonable_data = data[~valid_mask]
traffic_event_counts = data["TrafficEvent"].value_counts()
gender_stats = data.groupby("Gender")[["Speed", "TravelDistance", "TravelTime"]].mean()
```

## 五、易错点纠正（高分关键）

- 字段字符串引号必须配对：`data["Age"]`，不能写成 `data['Age]`  
- 变量名前后一致：`pa_s` 不能误写成 `ps_s`  
- `np.inf` 不能写成 `np_inf`  
- 删除列用 `drop(columns=[...])`，不是 `drop(column=...)` 或 `drop[colums=...]`  
- `duplicated()` 拼写要正确，不是 `dumplicated()`  
- 导出是 `to_csv("xx.csv", index=False)`，别漏 `index=False`  
- 多条件筛选必须每个条件加括号，并用 `&` / `|` 连接

## 六、考场 10 分钟执行清单

1. 先 `read_csv + head()`，确认字段名是否与题干一致

5. 所有辅助列在导出前删除

7. 检查是否需要截图、截图命名是否符合要求

## 七、一页速背（最后冲刺）

- 主流程：**读取 -> 清洗 -> 审核/打标 -> 统计 -> 分箱 -> 导出**

- 三大审核：`between`、阈值比较、`all(axis=1)`  
- 三大清洗：`dropna`、`fillna(ffill+bfill)`、`astype`  
- 三大输出：打印关键结果、保存清洗表、按要求截图
